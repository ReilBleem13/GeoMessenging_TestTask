## Установка CloudPub

1. Установить `wget https://cloudpub.ru/download/stable/clo-3.0.1-stable-linux-x86_64.tar.gz`
2. Разархивировать `tar -xvf clo-3.0.1-stable-linux-x86_64.tar.gz`
3. Переместить `sudo mv ~/clo /usr/local/bin/clo` и дать права `sudo chmod +x /usr/local/bin/clo`
4. Залогинится через сайт с помощью `clo login`
5. Или можно использовать ключ (взятый на сайте) `clo set token YOUR_KEY`

Все, можно пользоваться. Вбив команду `clo` можно посмотреть доступные команды
